O que é a Linguagem C?

Criada nos anos 70, a linguagem C é a base de muitos sistemas operacionais (como o Linux e Windows) e outras linguagens modernas. Ela é poderosa, rápida e excelente para entender como o computador funciona "por baixo do capô".

Seu Primeiro Programa

Todo programador começa imprimindo uma mensagem na tela. Veja o famoso "Hello World":

#include <stdio.h>

int main() {
    printf("Olá, Mundo!\n");
    return 0;
}

Dica: O #include <stdio.h> serve para que possamos usar a função printf.

Variáveis e Tipos

Em C, você precisa dizer ao computador que tipo de dado está guardando:

  • int: Números inteiros (ex: 10, -5).
  • float: Números com vírgula (ex: 10.5).
  • char: Um único caractere (ex: 'A').

Tomando Decisões (if/else)

Usamos o if para testar se algo é verdadeiro:

int idade = 18;

if (idade >= 18) {
    printf("Você é maior de idade.\n");
} else {
    printf("Você é menor de idade.\n");
}
